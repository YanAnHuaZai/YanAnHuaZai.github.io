## Java中的基本数据类型有：


| 类型\字节数 | 1字节 | 2字节 | 4字节 | 8字节 |
| :--: | :--: | :--: | :-: | :--: |
| 整数 | byte | short | int | long |
| 布尔 | boolean |       |       |      |
| 字符 |  | char | | |
| 浮点数 |  | | float | double |

byte short int long (整数)

boolean (布尔)

char (字符)

float double (浮点数)
